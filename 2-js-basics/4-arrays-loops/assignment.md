# Loop an Array

## Instructions

Create a program that lists every 3rd number between 1-20 and prints it to the console.

> TIP: use a for-loop and modify the iteration-expression
> 
 https://codepen.io/Blahovistna/pen/eYKmLqP?editors=0012

```
for (i=1; i<=20; i++) {
  if (i%3==0) {
  console.log(i);
  }
}
```
or
```
for (i=0; i<=20; i+=3) {
  console.log(i);
}
```
